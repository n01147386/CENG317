    Build Instructions

    Introduction


    The goal of this project is to automatically capture license plate when a car is entering or exiting the parking lot, after the the system read the license plate successfuly, it will send the number to my database and open thegate to let the car in or out. this will be a hands-free parking lot. This system is connected to the database, and our user will be able to pay the parking fee on their smartphone with their own account. The current progress of this project is to simulate how the parking lot gate works. This is done with the ultrasonic sensor, motor driver and raspberry pi. The sensor will trigger the motor to open the gate and turn on the green light if it detecte a car coming closer to the gate, and close the gate, turn on red light if there is no more object in front of the gate.
